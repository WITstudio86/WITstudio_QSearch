# 约束依赖

```
T TraitA:TraitB
```
> 指的是一个类型在实现 TraitA 的时候必须要以实现 TraitB 为前提
> 反之不成立


```rust
fn main() {
    struct TypeA;
    struct TypeB;

    trait TraitA {}
    trait TraitB:TraitA {}

    impl TraitA for TypeA {}
    impl TraitB for TypeA {}

    impl TraitB for TypeB {}// //error[E0277]: the size for values of type `TypeB` cannot be known at compilation time
}
```

```rust
fn main() {
    println!("{}", -3.1415926535);
    println!("{}", 2.);
    println!("{}", 2e2); // 200
}
```

## 常量

### 特殊值

- `INFINITY` 正无穷 , `NEG_INFINITY` 负无穷
- `MIN` 最小有限量
- `MAX` 最大有限量
- `NAN` 非数值

```rust
fn main() {
    println!("{}", f64::MAX);
    println!("{}", f64::MIN);
    println!("{}", f64::INFINITY);
    println!("{}", f64::NEG_INFINITY);
}
```

### 数字常量

std::f32::consts 模块和 std::f64::consts 模块提供了各种常用的数学常量，比如 E、PI 和 2 的平方根。

```rust
fn main() {
    println!("{}", std::f64::consts::PI);
}
```

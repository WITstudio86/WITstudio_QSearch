# 最小配置

## user 信息

```bash
$ git config --global user.name 'your name'
$ git config --global user.email 'your email@domain.com'
```

## 作用域

- `--local` 只针对当前仓库
- `--global` 针对当前用户的所有仓库
- `--system` 对系统所有用户生效

> 缺省表示 local


## 查看当前配置

```bash
$ git config --list --local
$ git config --list --global
$ git config --list --system
```
